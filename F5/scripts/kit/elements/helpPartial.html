Help classes

Margins

  • no-margin

    Remove all margins from element

  • margin-*

    Use it to change element margins. Available *: 8, 16, 32, 64

  • margin-top-*

    Use it to change top margin. Available *: 0, 8, 16, 32, 64

  • margin-bottom-*

    Use it to change bottom margin. Available *: 0, 8, 16, 32, 64

  • margin-left-*

    Use it to change left margin. Available *: 0, 8, 16, 32, 64

  • margin-right-*

    Use it to change right margin. Available *: 0, 8, 16, 32, 64

Padding
