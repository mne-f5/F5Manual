Motion

Motion matters

Motion is used in applications to describe functionality , fluidity and relashionships between components.

Getting started

Motion design can be used in an app to improve user experience and it provides:

  • Focus between views
  • Hints the user with what a certain gesture will do
  • Provides users with useful feedback
  • Direct the user`s focus

Orientation

Direction is very important in motion because it gives users a sense of orientation and helps the user understand the change that happened in the layout and what triggered the change. A basic example is the hamburger menu with a side menu sliding from the left. Pressing the button will fluidly show a side menu quickly comming in from the left area. This will help the user associate the hamburger button with the side menu on the left and he will remember faster how to get to that menu.

Avoid surprising the user , orient him.

Animations should be fast enough and directional

Animations should convey succes and failure

Animation methods

Modern browsers can animate four styles making use of the transform and opacity properties.

  • Position — transform: translateX(n) translateY(n) translateZ(n);
  • Scale — transform: scale(n);
  • Rotation — transform: rotate(ndeg);
  • Opacity — opacity: n;

Avoid using transitions with left/top/right/bottom
