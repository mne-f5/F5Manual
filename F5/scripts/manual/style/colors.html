Colors

Color combinations

Colors inside an app can suggest interactivity and can provide visual consistency. Choosing a harmonious color pallete can bring a great impact on the user experience.

Theory

The apps color palette should be defined by a custom palette suited to your brands identity. All color palletes should include sufficient contrast between different UI components.

Using complementary colors - Colors should work together and not create conflict or distract.

Choose a color pallete coordinated with your app logo .

Primary color

Primary colors should be widely used across all components and screens. In our case it`s dark grey.

Secondary color

The secondary colors should indicate actions and information. The secondary color can be darker or lighter than the primary color.

Accent color

Accent color represents in fact the secondary color. This color can be used in :

  • Text fields
  • Text selection
  • Progress bars
  • Sliders , buttons , controls
  • Links
