Icons

System icons

Icons should be used to help users identify faster what the associated action represents.

Getting Started

Icons

A system icon can represent a file , a command and actions like save , print , delete etc. Icons should be simple , modern and friendly.

Icon sizing

Icons should be displayed at 24 points respecting the 8 point grid and mantaining accuracy. Not respecting this sizing can cause an icon to be fuzzy.

Colors

Default color for an icon on a light background should be 54% opacity of black (#000000) . An inactive icon should have an opacity of 26%.

Default color for an icon on a dark background should be 100% white (#FFFFFF) . Inactive icon should have a 30% opacity of white.
